## 🏗️ Architecture

- **Single HTML file** — 21,136 lines, 830+ KB
- **Zero dependencies** — No frameworks, no npm, no build step
- **Procedural art** — All visuals via Canvas 2D API
- **Synthesized audio** — All sounds via Web Audio API
- **localStorage** — Persistent progression between runs
- **60 FPS** — Performance budget enforced
